# 3D Runner Game

A 3D endless runner game built with Three.js that works on both web and mobile devices.

## Features

- **3D Graphics**: Full 3D environment with shadows and lighting
- **Cross-Platform**: Works on desktop and mobile devices
- **Endless Runner**: Infinite procedurally generated obstacles and coins
- **Progressive Difficulty**: Speed increases as score grows
- **Mobile Controls**: Touch-friendly interface for mobile devices

## How to Play

1. Open `index.html` in your web browser
2. **Desktop**: Use A/D or arrow keys to move, SPACE to jump
3. **Mobile**: Use the control buttons at the bottom
4. Avoid red obstacles and collect gold coins
5. Score increases over time and with collected coins
6. Game speed increases as your score grows

## Game Mechanics

- **Movement**: Left/right movement with momentum
- **Jumping**: Jump over obstacles with realistic gravity
- **Scoring**: 
  - Continuous score based on distance traveled
  - +10 points for each coin collected
  - Speed multiplier affects scoring rate
- **Difficulty**: Speed increases every 100 points

## Controls

### Desktop
- **A/D Keys** or **Arrow Keys**: Move left/right
- **SPACE**: Jump
- **SPACE/ENTER**: Restart after game over

### Mobile
- **← Button**: Move left
- **→ Button**: Move right  
- **JUMP Button**: Jump
- **Tap screen**: Restart after game over

## Technical Features

- Three.js 3D rendering
- Real-time shadows and lighting
- Responsive design for all screen sizes
- Touch controls for mobile devices
- Smooth physics and collision detection
- Procedural obstacle and coin generation
